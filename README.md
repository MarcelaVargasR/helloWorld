# Hello World Project

## Overview

This software is a basic web page built using HTML. When opened in a browser, it displays the message “Hello World” using a heading element. The page follows standard HTML structure, including the document type declaration, head metadata, and body content.
The purpose of creating this software was to practice organizing a project folder, writing a properly structured HTML document, using Git for version control, and publishing the project to GitHub.
